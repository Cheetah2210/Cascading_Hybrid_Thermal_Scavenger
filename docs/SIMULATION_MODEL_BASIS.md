# Simulation Model Basis: 4-Stage Solid-State Cascade

## 1. Thermodynamic Framework
The GEN-III CHTS operates on a sequential thermal extraction model:
- **Stage 1 & 2 (High/Low TEG):** Solid-state Seebeck conversion (18.5% + 14.0% efficiency).
- **Stage 3 (Zeotropic):** Binary fluid heat recovery (7.1% efficiency).
- **Stage 4 (Adsorption):** Silica Gel-Water desiccant cycle providing a 5.0% cooling offset.

## 2. Total Efficiency Target
The system is modeled at a cumulative conversion efficiency of:
$$\eta_{\text{total}} = 0.185 + 0.140 + 0.071 + 0.05 = 0.446 \text{ (44.6%)}$$

## 3. Assumptions
- Input is treated as pure sensible thermal energy.
- No electromagnetic field losses or ionized flux calculations are required.
- Saturation point is locked at 200kW input capacity.
