# CHTS Industry Scaling & Applications

This system is designed for high-exergy thermal reclamation.

- **Geothermal Mining:** Direct integration with supercritical brine loops to recapture blow-off thermal energy.
- **Industrial Smelting:** Exhaust stack integration to scavenge energy from high-temperature off-gas.
- **Aerospace:** High-altitude turbine waste-heat recovery to improve net propulsion efficiency.

*Note: Scaling factors for these applications are determined by the `calculate_cascade()` function within `variables/zeotropic_mix.py`.*
